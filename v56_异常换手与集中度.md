# V56 — 行业资金热度 v2: 异常换手(去规模偏) + 成交集中度(资金聚焦)

> 用户: "你刚才计算的行业资金热度具体用的什么方法, 我感觉资金聚焦成交上升的热点行业会有帮助啊。"
> 背景: V55 用 `amt`(行业总成交额) / `amtr`(成交额5日变化) 测资金热度, OOS 弱(3.83~3.96)。用户指出直觉未被真正检验——V55 方法有两个缺陷: (1) 总成交额被行业**规模**绑架; (2) 只测**聚合总量**, 没测**聚焦/集中度**。本篇用修正方法重测。
> 承接 V55: 修复规模偏与无集中度缺陷, 对齐"资金聚焦成交上升的热点行业"。

---

## 1. 修正后的信号构造(均行业横截面 z)

| 信号 | 构造 | 修复的 V55 缺陷 |
|---|---|---|
| `abn` 异常换手 | 行业成交额 / 自身 MA20 − 1 | 去规模偏(比自身常态更热, 非"总量最大") |
| `conc` 成交集中度 | HHI归一化 = (Σ(个股权额占比)² − 1/N) / (1 − 1/N) ∈[0,1] | 去行业成员数偏 + 建模"资金聚焦龙头" |
| `abn_conc` 组合 | z(abn) + z(conc) | 异常×聚焦 相加(正交信号不双重计数) |

> 成交额 = 成交量 × 收盘(各成员), 与 V55 同数据源; HHI归一化消除"小行业天然高集中度"的规模偏。

## 3. 正交性诊断(候选池横截面 n=174,014, 与 slope / avgdif 相关性)

| 信号 | corr(信号, slope) | corr(信号, avgdif) | 判读 |
|---|---:|---:|---|
| abn | −0.015 | **+0.297** | 与 avgdif 冗余(量在价后放大=成交量动量代理) |
| conc | −0.012 | −0.011 | **最正交**: 独立於斜率与 avgdif, 真正"资金聚焦"切面 |
| abn_conc | −0.017 | +0.188 | 受 abn 冗余拖累 |

- `conc` 是迄今测试的**唯一同时独立于个股斜率与 avgdif** 的行业信号 —— 证明"资金聚焦龙头"这一切面被**正确、独立地建模**了出来。
- `abn` 与 avgdif 相关 +0.297, 印证"异常换手"本质仍是趋势/动量代理, 非正交增量。

## 5. 结论(诚实)

1. **用户的"资金聚焦"直觉被方法正确建模了**: `conc`(HHI归一化集中度) 是测试史上最正交的行业信号(corr(slope)=−0.012, corr(avgdif)=−0.011), 真正捕捉"钱往龙头集中"切面, 与趋势完全独立。
2. **但即便最正交的信号, OOS 仍低于基线(4.06 < 4.54) 且 0/4 胜基线** —— 正交 ≠ 有预测力。在 中证1000 / T+1开盘 / 2%仓位 这一设定下, "资金聚焦成交上升的热点行业"未能转化为样本外 alpha。
3. **V55 的弱结论不是"测得太粗"的假象**: 修正规模偏(HHI归一化)+ 显式建模集中度后, 结果仍弱。说明该域信号确实无效, 而非测量artifact。
4. **行业信号源搜索在 OHLCV 域内已彻底穷尽**: V54(avgdif/indmacd/fwdret) + V55(amt/amtr/breadth/breadth_d) + V56(abn/conc/abn_conc) 覆盖趋势/量价/集中度全谱。可交易的均 OOS ≤ 基线 或 << avgdif; 唯一强信号 fwdret 含未来信息不可交易。**avgdif k=0.26(32.30 / OOS 5.33, 3/4) 仍为最佳可交易行业倾斜。** 该线程正式关闭。
